# Task Management System

A full stack web application for managing tasks built using Angular and ASP.NET Core Web API.

## Features
- Create tasks
- View tasks
- Update tasks
- Delete tasks
- REST API integration
- SQL Server database

## Technologies Used
- Angular
- ASP.NET Core Web API
- C#
- SQL Server
- Entity Framework Core
- REST API

## Project Architecture
Angular Frontend → ASP.NET Core API → SQL Server Database

## How to Run

### Backend
Run the ASP.NET Core API using Visual Studio.

### Frontend
cd task-manager-ui
ng serve

Open:
http://localhost:4200
